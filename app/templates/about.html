What is SecureHide?

A robust system combining advanced cryptography with multimedia steganography

SecureHide is an innovative platform that combines cryptography (AES/RSA encryption) with multimedia steganography (image, audio, video) to create a secure channel for secret communication.

It allows users to encrypt messages and embed them invisibly into media files, ensuring confidentiality, integrity, and secrecy in a single framework. While encryption protects the contents of a message, it does not conceal its existence. Steganography, on the other hand, hides the very presence of communication.

Key Features

  •  Strong Encryption: AES-256 for message security with optional RSA for key exchange
  • Multiple Media Support: Hide data in images, audio, and video files
  •  Integrity Verification: SHA-256 hashing ensures messages haven't been altered
  • User Authentication: Secure login system protects your data
  •  File Management: Track and organize your stego files
  •  Flexible Implementation: Web-based interface for easy access

Technical Details

Encryption
  • AES-256-GCM encryption
  • RSA-2048 for key exchange
  • PBKDF2 key derivation
  • Random salts for extra security
Steganography
  • LSB (Least Significant Bit) technique
  • Image steganography (JPG, PNG)
  • Audio steganography (WAV, MP3)
  • Video steganography (MP4, AVI)
Web Platform
  • Flask web framework
  • SQLite/SQLAlchemy database
  • Bootstrap 5 responsive design
  • Secure password hashing
